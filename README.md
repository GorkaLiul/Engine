# Engine
 Modular extension based 3d rendering engine, **USES:** ImGui Cmake GLFW GLAD (linux compatible)


## Instalation: 
 run in terminal: 
```
 git clone https://github.com/GorkaLiul/Engine.git
```
## Build:
```
   mkdir build
   cd build
   cmake ..
   make
```
## Uninstall: 

```
cd ~
rm -rf Engine
```
